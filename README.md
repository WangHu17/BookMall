# 开发环境
- 数据库：MySql 8.0.22
- 服务器：apache-tomcat-9.0.41
- 编译器：IntelliJ IDEA 2021.1.1
- 浏览器：Chrome
# 技术栈
java、sql、servlet、jsp、jdbc、jstl、html、css、JavaScript、jQuery
# 功能
- 登录、注册、验证码、分页、权限管理（普通用户和管理员）、查询图书、图书管理（管理员-添加、修改、删除）、购物车（session）、订单管理（查看详情、管理员-发货、签收）
